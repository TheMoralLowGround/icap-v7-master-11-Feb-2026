This is the module that handles key-view keys.

====Key Module Central====
Script that uses method scirpts(Anchors/Regex/Keys/Single Column Extractor etc scripts).
Primarily gets text and position and text, Page ID for a given field.



====Key Extractor Function====

Primary consolidator of keys extracted using regular key module central scripts and robot files.
To generate keyNode structure.



====Key Children Appender====
Script that creates children inside each keynode for key-view.
Used by frontend to display compound/address type fields.
